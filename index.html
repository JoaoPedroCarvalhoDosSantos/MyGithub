Conceitos de GIT e GitHub

O Git possibilita a integração de fluxos de trabalho realizados por vários contribuidores com o passar do tempo, para um determinado repositório. O GitHub é um serviço de hospedagem na Web para repositórios do Git.

Fluxo do Git / GitHub

O fluxo Git/GitHub envolve usar o Git localmente para controlar as versões dos arquivos e, em seguida, usar o GitHub como um repositório remoto para compartilhar e colaborar no código.
